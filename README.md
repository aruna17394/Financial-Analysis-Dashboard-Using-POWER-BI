# Financial-Analysis-Dashboard-Using-POWER-BI
This Power BI dashboard is a Financial Data Analysis Dashboard, designed to provide insights into budget, expenses, profit/loss, and transaction trends. 
 Here’s a breakdown of the key sections:

Key Metrics (Top Row)
Total Budget – 1236K: The allocated budget for a specific period.
Actual Expense – 1331K: The amount spent so far.
Total Profit/Loss – -95093: A negative value indicates a financial loss.
Avg Profit % – -42.80%: Represents the profitability percentage, showing losses.
# Transactions – 450: The total number of financial transactions recorded.
Visual Breakdown of Data
Sum of Actual Expense by Category (Left Pie Chart)
Breakdown of actual expenses across categories such as Salaries, Rent, Miscellaneous, Operations, and Marketing.
Percentages indicate the proportion of each category in the total expense.
Sum of Budget by Category (Right Pie Chart)
Shows how the budget is allocated across different categories.
# Transactions by Profit/Loss Status (Pie Chart, Right Middle)
Displays the proportion of profitable vs. loss transactions.
207 transactions resulted in a loss (46%), while 243 transactions were profitable (54%).
Actual Expense and Budget by Month (Line Chart, Bottom Left)
Compares actual expenses against the budget for January, February, and March.
Helps track monthly spending trends.
Profit/Loss by Category and Status (Bar Chart, Bottom Middle)
Green bars represent profit, and blue bars represent loss in different categories.
Helps identify which categories contribute most to financial gains or losses.
# Transactions by Day and Profit/Loss Status (Bar Chart, Bottom Right)
Tracks the number of transactions each day, categorized by profit or loss.
Useful for understanding daily financial trends.
Filters (Right Sidebar)
Profit/Loss Status Filter: Allows selection between Profit, Loss, or All transactions.
Category Filter: Enables filtering of financial data by specific categories like Marketing, Miscellaneous, Operations, Rent, and Salaries.
Purpose of the Dashboard
Helps businesses track and analyze their financial performance.
Identifies areas of overspending and underperformance.
Assists in decision-making to improve financial efficiency.
